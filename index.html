Girdle

Girdle is a collection of experiments using force directed input to control a horizontal scroller.

Github

https://github.com/foldi/Girdle

Compatibility

All experiments are compatible in the latest version of Chrome. Some use the Leap Motion device (https://www.leapmotion.com/).

Examples

Girdle Mouse

Control the scroller using the mouse's proximity from the center of the screen.

Girdle Leap

Control the scroller using a Leap Motion device.
